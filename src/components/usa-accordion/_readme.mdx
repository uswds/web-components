# Accordion

A accordion component hides or reveals hidden content panels when selected.

- [Learn more about using this component on designsystem.digital.gov](https://designsystem.digital.gov/components/accordion/)
- [Accessibility tests](https://designsystem.digital.gov/components/accordion/accessibility-tests/)
- [Release notes]

## Element name
`<usa-accordion>`

## Implementation example

<Canvas of={AccordionStories.OpenPanel} />

## Expected elements
| Name | Description | Example code | Count |
|:--------|:--------|:--------|:--------|
| `<details>` | Wrapper | `<details></details>` | 1+ |
| `<summary>` | Panel title |`<summary>[Title text]</summary>` | 1 per `<details>` element |
| `<div slot="accordion-body">` | Panel content |`<div slot="details-body">[Collapsible panel content]</div>` | 1 per `<details>` element |

## Props and attributes

| Attribute/prop | Element | Description |
|:--------|:--------|:--------|
| `multiselect` | `<usa-details>` | Allow multiple panels to be open at once.
| `bordered` | `<usa-details>` | Add a border to the details element.
| `open` | `<details>` | Add the `open` attribute to a child `<details>` element to make the content inside the `details-content` slot visible on page load.

## Parts

| Part | Description |
|:--------|:--------|
| `wrapper` | The base `<details>` element
| `summary` | The `<summary>` element inside of `<details>`
| `content` | The content panel inside of `<details>`
